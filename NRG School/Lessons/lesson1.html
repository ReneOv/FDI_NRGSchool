Introduction to HTML

Theory

HTML (Hyper Text Markup Language) is a language for specifying how text and graphics appear on a web page.
HTML is not case sensitive language.
HTML documents contain HTML tags and plain text.

Elements and Tags

A tag is always enclosed in angle brackets < and >.
HTML tags usually come in pairs. An opening tag < HTML >, and the closing tag < /HTML >

Main tags


<html> - Describe HTML web page that is to be viewed by a web browser.
<head> - This defines the header section of the page.
<title> - This shows a caption in the title bar of the page.
<body> - This tag show contents of the web page will be displayed.



Types of tags
