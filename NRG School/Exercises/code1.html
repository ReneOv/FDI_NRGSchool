Practice Introduction to HTML

Welcome to your first website!

Instructions


1. Create a new folder in My Documents.
2. Make a new folder named Lab1 in the previously created folder.
3. Create a new Web page called Lab1.html in the Lab1 folder



a) Headings, paragraphs and rules


  • Create a heading h1 with the title: "Lab 1. Introduction to HTML". The title must be centred with the font type and colour you choose.
  • Add a line break <br>
  • Create two headings <h2> with the names of your hobbies or favourite free time activities.
  • Add a description about your hobbies under each heading. The description of each hobby must be in a paragraph <p>.
  • Modify the font type and colour of the headings of your hobbies and your descriptions using the <font> tag.
  • Add a horizontal rule <hr>.


    • Ex. 

    <h1 align = "center"><font color = #cc9900 face = "verdana">
	Lab 1. Introduction to HTML
</font></h1>
<br>
<h2><font color = #cc9900 face = "verdana">
	Hobby 1
</font></h2>
<p align = "justify"><font face = "courier new">
	Hobby 1 description.
<br><br>
</font></p>
<h2><font color = #cc9900 face = "verdana">
	Hobby 2
</font></h2>
<p align = "justify"><font face = "courier new">
	Hobby 2 description.
</font></p>
<hr>
